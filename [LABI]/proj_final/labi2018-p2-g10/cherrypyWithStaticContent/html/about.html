Sobre o trabalho

Este trabalho foi nos proposto pela unidade curricular de Laboratório de Informática,numa primeira fase tentamos abordar o projeto de uma maneira integral, mas com o passar do tempo,devido à dependencia de cada módulo, não obtivemos muitos progressos.Por isso decidimos dividir o trabalho em módulos independentes, como está explicito na 'sidebar' desta página.

O site está dividio em 4 subpáginas,onde as primeiras 3(Home,Listagem e Aplicação), são as pedidas no guião do projeto. Acrescentamos apenas esta página, para dar uma pequena explicção do trabalho.

Para o layout do site utilizamos um template retirado do website:https://www.os-templates.com/free-website-templates

Por fim,apesar do nosso projeto não funcionar integralmente, agradecemos a disponibilidade dos professores desta unidade curricular para este projeto.
